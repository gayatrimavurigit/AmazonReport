# Power BI Sales Dashboard

This project is an interactive Power BI dashboard created using Excel data to analyze sales performance.

## Tools Used
- Power BI
- Excel

## Features
- Interactive dashboard
- Filters and slicers
- KPIs and charts

## Insights
- Identified top-performing products
- Analyzed sales trends across categories

## Files Included
- .pbix file (Power BI report)
- Excel dataset
- Dashboard screenshots
